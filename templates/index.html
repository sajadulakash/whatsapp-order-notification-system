API Endpoints (for WhatsApp Bot)

GET /api/orders/pending
Get all orders with pending (non-completed) status
GET /api/orders/{order_id}
Get a single order by ID
GET /api/orders/{order_id}/responsible-users
Get users responsible for the current stage of an order
POST /api/notifications/send/{order_id}
Send WhatsApp notifications to all responsible users for an order
POST /api/notifications/confirm/{notification_id}
Mark a notification as confirmed/acknowledged
GET /api/notifications/{order_id}
Get all notification records for an order
GET /api/whatsapp/log
View dummy WhatsApp message log (debug)
